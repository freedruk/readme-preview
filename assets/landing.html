Docs that don't break on npm.

Preview your README, rewrite relative assets (and optionally links), and enforce doc quality in CI — before you publish.

npx readme-preview npx readme-preview check --strict npx readme-preview init --assets
Open on npm

What it does

  • Preview README rendering locally with npm or GitHub theme options
  • Rewrite relative images and links so they work outside your repository
  • Check documentation quality in CI with strict validation mode
  • Init scaffolds GitHub Action workflow, badges, and screenshot section automatically
